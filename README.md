# Meme API Tests

Проект по автоматизированному тестированию REST API для работы с мемами.

Тестируется API:
http://memesapi.course.qa-practice.com

---

## Используемые технологии

- Python
- pytest
- requests

---

## Что тестируется

### Позитивные сценарии
- Создание мема
- Получение мема по id
- Проверка сохранённых данных
- Получение списка мемов
- Обновление мема
- Удаление мема

### Негативные сценарии
- Получение несуществующего мема (404)
- Обновление несуществующего мема (404)
- Удаление несуществующего мема (404)
